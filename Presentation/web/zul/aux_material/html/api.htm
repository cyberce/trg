Δημιουργία Αγγελίας

Για τη δημιουργία αγγελίας από ένα τρίτο σύστημα καλείται η ακόλουθη μέθοδος:

url

https://swaps.teiath.gr/services/product

method

POST

 

request body

NAI

 

παράμετροι

OXI

 

αυθεντικοποίηση

NAI

 

 

Η παραπάνω μέθοδος λαμβάνει τo παρακάτω αρχείο XML

 

<?xml version="1.0" encoding="utf-8" ?>

<product>

<productName>Samsung S4</productName>

<description>smartphone</description>

<purchaseDate>2013-12-25T16:00:00+03:00</purchaseDate>

<productBrand>Samsung</productBrand>

<price>600.00</price>

<currency>1</currency>

<sendHome>true</sendHome>

<enabled>true</enabled>

<comments>comments</comments>

<transactionType>1</transactionType>

<productCategory>1</productCategory>

<productStatus>1</productStatus>

</product>

 

Επίσης μπορεί να λάβει το παρακάτω αρχείο JSON

 

{

"productName": "Samsung S4",

"description": "smartphone",

"purchaseDate": "2013-12-25T16:00:00+03:00",

"currency": "1",

"sendHome": "true",

"enabled": "true",

"productStatus": "1"

}

Διάθεση νέου Περιεχομένου της Υπηρεσίας μέσω RSS feeds

 

Το νέο περιεχόμενο που προστίθεται στην Υπηρεσία ταξινομείται χρονικά και διατίθεται μέσω της πηγής RSS. Ο σύνδεσμος της πηγής RSS διατίθεται στην ιστοσελίδα της Υπηρεσίας ώστε οι χρήστες να έχουν τη δυνατότητα περαιτέρω αξιοποίησης της πηγής (χρήση σε RSS readers κ.λ.π).
